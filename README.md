# mysql-express-api

mysql-express-api is a Node.js API for CRUD operations on MySQL.

## Clone Repo

```bash
git clone 
```

## Installation

Use the package manager [npm](mysql-express-api) to install dependencies. Go to directory and follow commands below:

```bash
npm install
```

## Usage

```bash
npm start
```
