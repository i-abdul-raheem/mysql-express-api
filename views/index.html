Getting Started

Welcome to my MySQL Express API. This is a simple to use Restful API created to perform CRUD operations on MariaDB. You can use POST to create row or send data, GET to read rows, PUT to update row and DELETE to delete row.

Try Now

CRUD - /api/users

Login - /login

Endpoints

There are two endpoint, one for CRUD operations and the other one for LOGIN.

  • /api can be used to perform CRUD operations.

  • /login can be used to login and get x-access-token.

Authentication

To use this API you need to login first and set x-access-token header to be able to send requests.

To login you need to have a Username or Email and Password.

JSON POST DATA

With Username:
{ "username": "YOUR USERNAME", "password": "YOUR PASSWORD" }

With Email:
{ "email": "YOUR EMAIL", "password": "YOUR PASSWORD" }

Request
POST - /login
Request Body
Field Value Type Required
username String true (if email not given)
email String (Email) true (if username not given)
password String true
Response

  {
        "id": 1,
        "fname": "FIRST NAME",
        "lname": "LAST NAME",
        "username": "USERNAME",
        "password": "PASSWORD",
        "email": "EMAIL",
        "token": "X-ACCESS-TOKEN"
  }

Get all table records

To get all records from a table you need to put "Table Name" in place of :table_name.

Request
GET - /api/:table_name
Headers
Field Value Required
x-access-token JWT Token true
Content-Type application/json true
Response

  [{
        "id": 1,
        "fname": "FIRST NAME",
        "lname": "LAST NAME",
        "username": "USERNAME",
        "password": "PASSWORD",
        "email": "EMAIL",
  }]

Get Single Record

To get a single record from a table you need to put "Table Name" in place of :table_name and "Record ID" in place of :id.

Request
GET - /api/:table_name/:id
Headers
Field Value Required
x-access-token JWT Token true
Content-Type application/json true
Response

  {
        "id": 1,
        "fname": "FIRST NAME",
        "lname": "LAST NAME",
        "username": "USERNAME",
        "password": "PASSWORD",
        "email": "EMAIL",
  }

Create Record

To create a record into a table you need to put "Table Name" in place of :table_name.

Request
POST - /api/:table_name
Headers
Field Value Required
x-access-token JWT Token true
Content-Type application/json true
Request Body
Field Value Type Required
Data JSON true
Response

  {
        "fieldCount": 0,
        "affectedRows": 1,
        "insertId": < INSERTED_ID >,
        "serverStatus": 2,
        "warningCount": 0,
        "message": "",
        "protocol41": true,
        "changedRows": 0
  }

Update Record

To update a record into a table you need to put "Table Name" in place of :table_name and Record ID in place of :id.

Request
PUT - /api/:table_name/:id
Headers
Field Value Required
x-access-token JWT Token true
Content-Type application/json true
Request Body
Field Value Type Required
Data to Update JSON true
Response

  {
        "fieldCount": 0,
        "affectedRows": 1,
        "insertId": 0,
        "serverStatus": 2,
        "warningCount": 0,
        "message": "(Rows matched: 1 Changed: 1 Warnings: 0",
        "protocol41": true,
        "changedRows": 1
  }

Delete Record

To delete a record from a table you need to put "Table Name" in place of :table_name and Record ID in place of :id.

Request
DELETE - /api/:table_name/:id
Headers
Field Value Required
x-access-token JWT Token true
Content-Type application/json true
